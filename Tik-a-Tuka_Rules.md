# 🎲 티카투카(Tik-a-Tuka) 최종 규칙 정리

**게임 방식:** 주사위를 굴려 각 줄에 배치하고, 알까기(카운터)를 활용해 상대보다 높은 점수를 채워 승리하세요!

### 1. 기본 진행
*   **보드:** 3줄(왼쪽: 본인, 오른쪽: 상대)로 구성되며, 각 줄은 주사위 3개까지 수용 가능합니다.
*   **턴 시작:** 자신의 턴에 주사위를 굴려 시작합니다.
*   **종료:** 양쪽 보드가 모두 꽉 차면 종료됩니다.

### 2. 점수 계산 (주사위 합계 + 페어 보너스)
줄 점수는 **[주사위 눈 숫자의 총합] + [보너스 점수]**입니다.

* **보너스 시스템 (페어 시스템):**
    * **투 페어(2개 일치):** 주사위 눈 숫자만큼 보너스 점수 부여
    * **트리플(3개 일치):** 주사위 눈 숫자 × 2만큼 보너스 점수 부여

**[보너스 예시]**
*   **눈이 3일 때:** 투 페어 3점 / 트리플 6점
*   **눈이 5일 때:** 투 페어 5점 / 트리플 10점
*   **눈이 6일 때:** 투 페어 6점 / 트리플 12점

### 3. 주사위 종류
*   **일반 주사위:** 자신의 보드에만 배치 가능.
*   **실드 주사위:** 자신의 보드와 상대 보드 모두 배치 가능. **알까기로 제거되지 않음.**

### 4. 공격 액션: 알까기(카운터)
상대 필드에 있는 주사위를 공격하여 제거합니다.
*   **조건:** 
    1. 굴린 일반 주사위의 눈이 상대 필드 해당 줄의 일반 주사위 눈과 같을 것.
    2. **자신의 해당 줄에 빈칸이 있어야 함.** (예: 1번 줄 알까기를 하려면 내 1번 줄이 꽉 차 있지 않아야 함)
*   **효과:** 상대 해당 줄의 눈 숫자가 같은 모든 일반 주사위를 제거합니다.
*   **보상:** 알까기 후 랜덤 숫자의 **실드 주사위**를 획득합니다.
    *   **주의:** 획득한 실드 주사위는 **반드시 획득한 그 턴 내에 사용**해야 합니다. (내/상대 보드 어디든 배치 가능)

### 5. 편의 및 진행 규칙
*   **리롤권:** 게임당 1회 사용 가능하며, 기존 숫자와 새 숫자 중 선택하여 배치 가능합니다.
*   **턴 건너뛰기:** 자신의 보드가 꽉 차면 자동으로 턴을 넘깁니다.

### 6. 승리 판정
1. **줄 승수:** 3줄 중 2줄 이상 승리 시 최종 승리.
2. **총점 합산:** 줄 승수가 같을 경우, 모든 줄의 총점 합산이 높은 쪽이 승리. (승수 우선)
